# 升级步骤

# 1. 升级 Astroboy

yarn add astroboy@3.0.0

# 修改应用启动方式

原来直接实例化 Astroboy 对象，即可完成应用的启动流程。3.X 将其拆为了两步：1.实例化应用对象，2.启动应用。需要将app/app.js改为下面的例子：

 const Astroboy = require('astroboy');

- new Astroboy();
+ const app = new Astroboy();
+
+ app.run()

另外，run().then(...)会在应用启动成功后执行。
